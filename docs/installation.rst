============
Installation
============


Stable release
--------------

To install particle_swarm_optimisation, run this command in your terminal:

.. code-block:: console

    $ pip install particle_swarm_optimisation

This is the preferred method to install particle_swarm_optimisation, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for particle_swarm_optimisation can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/invisible23man/particle_swarm_optimisation

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/invisible23man/particle_swarm_optimisation/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/invisible23man/particle_swarm_optimisation
.. _tarball: https://github.com/invisible23man/particle_swarm_optimisation/tarball/master
